nobr

The nobr tag is creates a text with non-breaking lines. It doesn't allow the text to divide multiple lines automatically. With the nobr tag the text is displayed in a single line, and a horizontal scroll bar appears to make it readable. Usually, if the text goes outside of the screen, the browser automatically break the text to the next line, but the nobr tag does not allow to break the line.
